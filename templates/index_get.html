Is it REAL?

Real Anot advises on the authenticity of news relating to COVID-19.
It uses Natural Language Processing techniques implemented via Neural Networks.

Check your facts!

Dataset

  • Subset of the CoAID dataset
  • Diverse COVID-19 healthcare misinformation dataset
  • 1,127 real and 266 fake news samples

Model

  • DistilBERT, faster and lighter-weight version of BERT
  • Tramsfer-learnt from pre-trained weights
  • Fine-tuned using CoAID dataset
  • Performed using the ktrain library

Test Results
